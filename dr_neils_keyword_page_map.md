# Dr Neil's Healthcare — Keyword-to-Page Map
### A focused SEO architecture, built to actually rank

> **Why this isn't 50 pages × 8 variations:** That structure violates Google's "Helpful Content" guidelines (doorway pages) and would likely *hurt* the site's rankings rather than help. Below is the architecture that works for a single-practitioner local health practice — 12 priority pages, ~60 keywords covered, each page targeting one clean search intent with 4–8 variations captured naturally.
> **How variations work in practice:** "homeopath Houston" and "homeopath in Houston" and "Houston homeopath" are all the same search intent to Google — one well-written page ranks for all of them. So below, the "variations" column shows the variations a single page will naturally rank for, not separate pages to build.
> **Total pages:** 12 core + 6 optional blog posts = up to 18 pages. **Total keywords covered: ~60–80.**

## 🟢 TIER 3 — LOCATION PAGES (build carefully)

> **Important — only build 2 of these to start, maximum 3.** Google penalizes near-duplicate "doorway pages" where a site builds /city-A, /city-B, /city-C with 90% identical content. **Each location page must have genuinely different content** — local references, drive-time language, neighborhood specifics, suburb-specific health concerns. If you can't write that much unique content for a suburb, don't make the page.

## 🗓️ BUILD ORDER & TIMELINE

If I had to build this from scratch, this is the order — by ROI, not by site map order:

### Month 1 (foundation)
1. Homepage `/`

4. Pricing `/pricing`

7. Allergies `/conditions/allergies` ← *single highest-ROI page for Houston*

### Month 3 (deepen the cluster)
10. Eczema `/conditions/eczema`

**Don't build everything at once.** Build 6 pages, watch the data, then build the next 3 based on what's actually getting clicks. SEO is an iterative game.

## 🟢 ONE LAST PRINCIPLE

**The page you build well outranks the keyword you target widely.**

A single deeply-written 2,000-word allergy page with real patient stories, specific Houston-climate insights, and an honest discussion of what homeopathy can and can't do for allergies — will outrank 10 thin pages stuffed with allergy keywords every time. This is what Google's last two years of algorithm updates have been about. Write fewer pages. Write them as if you're explaining the topic to a friend. Let the keywords land naturally inside good writing.
